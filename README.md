# Resend with Nuxt

This example shows how to use Resend with [Nuxt](https://nuxt.com).

## Instructions

1. Define environment variables in `.env` file.

2. Install dependencies:

  ```sh
npm install
# or
yarn
  ```

3. Run Nuxt locally:

  ```sh
npm run dev
  ```

4. Open URL in the browser:

  ```
http://localhost:3000/api/send
  ```
